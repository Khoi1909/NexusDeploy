# NexusDeploy
Một nền tảng PaaS (Platform-as-a-Service) tùy chỉnh, giúp biến mã nguồn GitHub thành ứng dụng web trực tiếp. Nền tảng tự động build &amp; test code trong Docker. Nếu build lỗi, AI sẽ phân tích log và gợi ý cách sửa. Nếu thành công, ứng dụng sẽ được tự động triển khai (host) lên một URL công khai với SSL.
